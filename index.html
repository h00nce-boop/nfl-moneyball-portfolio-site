01 / Project thesis

The edge is value, not just talent.

NFL teams do not only need great players. They need players whose production is worth more than their current price. This project turns that idea into a model, an interactive candidate board, and a validation story.

01

Find production before it gets expensive.

Rookie-contract players can create huge value because their cost is set before the market fully knows how good they are.

02

Compare performance to cost.

The core idea is simple: if a player ranks better in production than in cost, he may be underpriced.

03

Validate the signal.

The model is backtested against a comparable baseline to check whether flagged players actually performed better the next season.

“We've got to use every piece of data and piece of information, and hopefully that will help us be accurate with our player evaluation. For us, that's our life blood.”

Billy Beane
Plain-English formula: value gap = cost ranking - production ranking. A positive gap means the player is producing better than his cost would imply.
02 / Early findings

What the model is showing so far.

The website should make the takeaway clear before anyone opens the repository: this is about roster value, contract timing, and whether the signal survives a basic next-season check.

🦅

Eagles Lead on Roster Efficiency

Philadelphia is the main case study because the roster combines expensive stars with enough underpriced contributors to stay efficient.

📉

Cost Catches Up Eventually

When a young player becomes a proven star, the market reprices him. The model is designed to look for value before that happens.

💡

Wide Receiver Looks Interesting

Early testing suggests low-cost wide receivers may be one of the clearest places to find production before it is fully priced.

🎯

Backtest Shows Lift

At the default screen, flagged candidates had a 57.9% next-season hit rate versus 47.6% for the baseline group.

📊

Not a Black Box

The site lets users move filters and thresholds so the model can be explored, questioned, and stress tested.

🛠️

Still Evolving

This is a living portfolio project. The public website tells the story; the repository holds the deeper technical details.

08 / Living model

This is a living model.

This project is ongoing by design. The current version shows the core model, the interactive website, and first-pass validation; future versions will improve prediction quality, data coverage, and the product experience.

Current / V4

Backtested surplus signal

Current release: interactive candidate board, team value stories, default threshold backtest, threshold sensitivity, season stability, and a lightweight dashboard preview.

Planned / V5

Better prediction layer

Improve the year-over-year test with fairer comparison groups, injury and availability context, snap-share context, and clearer training/testing windows.

Planned / V6

Prospect-to-pro model

Move the model closer to draft season by connecting draft position, rookie-contract economics, college/Combine indicators, and early NFL production to identify potential breakout candidates sooner.

Future

Decision-support polish

Add cleaner player reports, automate data refreshes, and keep the public website focused on the story while the repository holds the deeper technical work.

Ongoing note: This is a living project. The website will continue to evolve as the model improves, new data becomes available, and the research moves from descriptive roster value toward earlier prediction.

This project was built as a portfolio piece demonstrating applied data science skills in a domain I care about: sports analytics and quantitative decision-making under uncertainty.

The methodology evolved across four versions — from a simple rank-gap model to a validated predictive framework with confidence flags, contract-cycle context, and first-pass backtesting.

Every design decision prioritized intellectual honesty: the model acknowledges its public-data limitations while still building toward a genuine decision-support tool.
